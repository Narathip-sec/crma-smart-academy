CRMA Smart Academy

Digital platform for cadets at Chulachomklao Royal Military Academy

Consolidates class schedule, military training, daily life management, announcements, grades, and AI-assisted fitness coaching into one app. Originally Expo + React Native; now pivoting to LIFF web app (Next.js + Prisma + PostgreSQL on Vercel). This document is the full handoff context for a fresh Claude session.

1 · Objectives

Problem

Cadets juggle fragmented systems for schedules, training, grades, mess hall, duty rosters, announcements. Paper-heavy, low transparency, no central feedback loop with units.

Solution

One mobile-first app: schedules, PFT results + AI coaching, mess hall menus, duty rosters (RBAC-gated), event creation, group chats, grades (past semesters only), portfolio.

Primary users

Cadets Instructors Officers Registrar

North-star metric

% of cadet workflows that no longer require paper / multiple logins. Secondary: time-to-acknowledge urgent announcements.

2 · Hard Constraints

🚫 LMS scope-out

Assignment submission stays on external Moodle https://lms.crma.ac.th/moodle/my/. Do not rebuild submission flow.

🔒 Current-semester grades LOCKED

Only past completed semesters viewable. Enforce in data layer, not UI.

👮 Duty roster RBAC

Roster mutations gated by rank/role. Cadets read-only, Officers+ can edit.

🔑 Auth gate

2FA required. @crma.ac.th email allow-list. Linked to cadet ID. No social login.

🇹🇭 Data sovereignty

Storage must reside in Thailand data centers. Rules out US-only managed services for primary data.

📋 PDPA + RTA classification

All access actions audit-logged. Compliance with PDPA B.E. 2562 + Royal Thai Army data rules.

9.1 · ServiceLink component (RN → port to web)

// app/src/components/ServiceLink.tsx — discriminated by kind
export type ServiceKind = "internal" | "external";

export interface ServiceLinkProps {
  id: string; kind: ServiceKind; labelKey: string;
  icon: LucideIcon; iconBg: string; iconColor?: string;
  onPress: () => void;
}

export function ServiceLink({ id, kind, labelKey, icon: Icon, iconBg, onPress }) {
  const { t } = useTranslation();
  return (
    <Pressable testID={`service.link.${id}`} onPress={onPress}
      className="mb-3 flex-row items-center rounded-2xl border border-slate-100 bg-white p-4">
      <View className={`h-10 w-10 items-center justify-center rounded-xl ${iconBg}`}>
        <Icon size={20} color={iconColor ?? "#0F172A"} />
      </View>
      <Text className="ml-3 flex-1 text-base font-bold text-slate-900">{t(labelKey)}</Text>
      {kind === "internal" ? <ChevronRight /> : <ExternalLink />}
    </Pressable>
  );
}
